Definition 1.2 - Binary Operation

A function  *  : S × S → S

Definition 1.3 - Groups

A pair (G,  * ) where G is a set and * is a binary operation, with the following axioms:

  • Associativity: (a * b) * c = a * (b * c), ∀a, b, c ∈ G

  • Identity: ∃e ∈ G : e * a = a = a * e, ∀a ∈ G. The identity is unique.

  • Inverse: ∀a ∈ G, ∃b ∈ G : a * b = b * a = e. The inverse is unique.

Definition 1.6 - Abelian Groups

A group (G,  * ) is abelian ⇔ its binary operation is commutative, namely, ∀a, b ∈ G, a * b = b * a

Definition 1.6 - Subgroups

H ≤ G ⇔ 1 ∈ H, ∀a ∈ H, a − 1 ∈ H and ∀a, b ∈ H, a * b ∈ H.

Namely, H ≤ G ⇔ a * b − 1 ∈ H

Definition 1.11 - Ring

Defined as a triple (R, +, ⋅ ) where R is a set with two binary operations.

  • Addition (R,  + ) forms an abelian group (since the addition binary operation is commutative)

  • Assocativity - The binary operation ⋅ is associative

  • Distributivity - Multiplication is distributive over additon

    • a ⋅ (b + c) = (a ⋅ b) + (a ⋅ c), ∀a, b, c ∈ R

  • Identity - There is a multiplicative identity, 1 ∈ R where 1 ⋅ a = a ⋅ 1 = a, ∀a ∈ R

    • 0 is the unique additive identity, and  − a is the unique additive inverse.

Lemma 1.14 - Ring Rules

  • ∀a ∈ R, a ⋅ 0 = 0

    • $a\cdot0=a\cdot(0+0)\overset{\text{distributivity}}{=}(a\cdot0)+(a\cdot0)$. Therefore, (a ⋅ 0) − (a ⋅ 0) = (a ⋅ 0) = 0 as required.

  • ∀a, b ∈ R, a ⋅ ( − b) =  − (a ⋅ b)

    • (a ⋅ ( − b)) + (a ⋅ b) = a ⋅ ( − b + b) = a ⋅ (0) = 0 ⇔ a ⋅ ( − b) =  − (a ⋅ b) as required.

Definition 1.20 - Integral Domain

Integral domain ⇔ commutative ring with 0 ≠ 1, and if a, b ∈ R and ab = 0 ⟹ a = 0 or b = 0. Namely, the cancellation property holds.

Division ring  ⟹ 0 ≠ 1 and every non zero element is a unit.

Definition 1.22 - Fields

A commutative division ring defines a field!

Definition 1.30 - Subrings

Definition 1.31 - Subrings

If S ⊆ (R,  + ) where R is a ring, S is a subring  ⇔ (S,  + ) satisfies three ring axioms (Addition, Associativity, Distributivity)

Definition 1.36 - Congruence

Let R be a ring and ∼ an equivalence relation on R.

Consider a, a′, b, b′ ∈ R where a ∼ a′ and b ∼ b′.

We call ∼ a congruence relation  ⟹ a + b ∼ a′ + b′ and a ⋅ b ∼ a′ ⋅ b′. The equivalence classes of a congruence relation are called congruence classes.

Theorem 1.37 - Congruence Rings

Suppose that ∼ is a congruence on a ring R. Consider the set R/∼ as the set of congruence classes where addition and multiplication is defined as follows:

  •  +  : [a] + [b] = [a + b]

  •  ⋅  : [a] ⋅ [b] = [a ⋅ b]

for a, b ∈ R.

Then, (R/∼,+, ⋅ ) forms a ring, with the zero element [0] and is commutative if R is commutative. (R/∼,+, ⋅ ) inherits a lot of the properties from (R, +, ⋅ )

Definition 1.38 - Ideals

Consider I ⊆ R where I ≠ ∅. I is an ideal  ⇔ I ≤ (R,  + ) where ∀a,  ∈ I and r ∈ R, ra ∈ I and ar ∈ I. Split closure property.

To check that I is an ideal:

  • a − b ∈ I where a, b ∈ I (To show that I is a subring)

  • ra ∈ I and ar ∈ I where a ∈ I and r ∈ R

Definition - Principle Ideal

Suppose that R is a commutative ring and let a ∈ R. Consider aR = {ar:r∈R} - this is an ideal in R and is called a principle ideal.
